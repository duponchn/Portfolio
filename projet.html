Création d’une base de données

Notre objectif dans ce projet était de créer une base de données pour un club de bowling. Nous devions créer toutes les tables et relations permettant une gestion des réservations dans le club de bowling.

Ce travail s'est effectué en trois étapes différentes expliquées ci-dessous

« Schéma relationnel / Script de création de la base de données / Test du script ». Dans la première étape nous devions créer le schéma de la base puis créer cette base et faire quelques requêtes dessus pour vérifier que tout fonctionne.

Classification automatique de dépêches — Algorithmes

Java · Algorithmique · Traitement de texte

2025

Création d’un système de classification de dépêches

Ce projet mène à des classifications de dépêches. Une dépêche c’est : un numéro, une date, une catégorie et un texte

Ce travail s'est effectué en trois étapes différentes expliquées ci-dessous

Pour chaque catégorie, nous avons créé un lexique manuellement avec des mots pour une catégorie avec une pondération allant de 1 à 3 (1 = mot peu fréquent, 3 = mot très fréquent). Puis créer un programme qui donne une catégorie aux dépêches selon les mots contenus dans les textes par rapport aux lexiques créés sur un fichier fourni.

Nous avons créer une génération automatique du lexique par rapport à des catégories connues. Cette classification ce fait en fonction du nombre de fois où apparaît le mot dans cette même catégorie comparée au nombre de fois que il appairaît dans les autres dépêches de catégorie différente . Ensuite nous avons pu classer les dépêches sur un fichier fourni.

Nous avons créer une génération automatique du lexique mais nous ne connaissons ni le nombre, ni le nom des dépêches sur un fichier fourni. Pour ceci nous avons donc dus d'abord récolter tous les thèmes des dépêches dans une liste, pour poursuivre nous avons due parcourir la liste et y créer les lexiques et enfin faire le classement générale des depêches.

Ce que j'ai fait dans ce travail

Ce projet a été fait en binôme et tout le travail a été fait ensemble sur le même ordinateur. Nous avons créé plusieurs fonctions pour répondre à des besoins. Un classement des dépêches, un calcul des scores, la génération automatique des lexiques, un système de tri pour trier les lexiquess…
